Checkout / shipping address form

Reproduces a guest-checkout address form using standard autocomplete address tokens.

Correct behavior: NO login autofill dropdown should appear here. This is address-autofill territory (a job for the browser's address autofill), not a login. Note the email field carries autocomplete="email" but there is NO password field in this form, so it must NOT be offered a saved login.

Shipping details

A typical guest checkout: name, address, city, postal code, email. autocomplete tokens like shipping address-line1 and email are present, but there is no password — this is not a login.
