Compose / DM / chat box

Reproduces messaging compose fields (Messenger, Slack, WhatsApp Web, X DMs).

Correct behavior: NO autofill dropdown should appear on any of these fields. No <form> and no password exist on this page.

1. Single-line message input

A formless type="text" with placeholder="Message..." and aria-label="Message" — the classic chat send bar.

2. Multi-line textarea compose

A <textarea> for longer messages/posts. The extension fills <input> usernames + password inputs only, but the dropdown must also never appear here.

3. Contenteditable compose div

A contenteditable div, as used by Slack/Messenger rich composers. Not an <input> at all, so it must never be treated as a login field.
