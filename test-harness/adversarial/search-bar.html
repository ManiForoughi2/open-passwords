Site search bar

Reproduces a typical e-commerce / docs header search input.

Correct behavior: NO autofill dropdown should appear on this field.

1. Formless search input (header)

A bare <input type="search"> with no surrounding <form> (see the nav bar above). Search is the single most common non-login text field on the web.

🔍

2. Search inside <form role="search">

Many sites wrap search in a form so Enter submits. The role="search" and the lack of any password field should keep autofill away.
